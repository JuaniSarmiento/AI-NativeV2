## MODIFIED Requirements

### Requirement: Activity student list includes report access
The teacher dashboard activity view SHALL include a "Ver Informe" action for each student with closed cognitive sessions, navigating to the report view.

#### Scenario: Docente navigates to report from student list
- **WHEN** a docente is viewing an activity's student list and clicks "Ver Informe" on a student
- **THEN** the system navigates to the report view for that student+activity combination
